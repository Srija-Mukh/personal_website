Technical Projects

Each technical project or assignment I have done has given me lots to learn. Here's where I reflect on what I achieved and learnt from some of my favourite technical projects.

Bash GitHub

April 2021

In this project, I was tasked with mimicking some of Git's functionality over a simple file system structure by writing shell scripts which would allow me to manipulate files. In order to complete the task successfully, I had to understanding Git's functionality and how stages of indexing and commiting work. This project opened my eyes to how software development may look in industry when the specification is limited and developers have to reserach and look into existing technologies to adapt functionalities. At the conclusion of this project I was able to implement initiating a git folder, adding, commiting and deleting files as well as creating and switching branches.

C2MIPS

May 2020

This was the project involved translating C to MIPS Assembly code. Although debugging was very hard, the task encouraged me to experience what low level programming looks like.

Minesweeper

April 2020

This was the project for my first ever programming course in university and involved replicating the popular 90s Microsot Windows game, Minesweeper in the C programming langauge. At the time 2D arrays were awfully confusing to me and so were functions. I wasn't so sure I would pass the assignment, given that the expectations of the assignment included both coding AND testing the program. After days of consistently coding and asking lots of questions to my tutor, I completed 3.5/4 stages of this assignment and was truly proud of myself.
